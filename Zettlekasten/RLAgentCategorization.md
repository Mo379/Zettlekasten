# RLAgentCategorization

- value based agents
	- No policy (implicit)
	- but has a value function
- Policy based agents
	- Policy
	- No value function
- Actor Critic agents
	- has a Policy
	- and has a Value function
 - Model free agent
	 - Policy and value function
	 - No model of the environemnt
- Model based 
	- has a Policy and or value function
	- Also has a model of the environemnt
